# 启动

```
npm install
```

```
npm start
```

```
npm run build
```

```
node app.js
```

# 温馨提示

1. 请不要改变目录中文件位置，以免运行出错。

2. 识别图片只支持 20\*20 大小的纯黑白单个数字图片（如 `data/test.png`）

# 使用方法

运行 `dist/index.exe` 或者 `npm start` 即可。

在命令行中，有以下可用命令：

- analysis 或者 a

对目录 `data/png` 中的图片进行特征分析，所有数据结果保存在 `data/json` 中。

- classify 或者 c

对图片进行分类（即识别数字）。

然后请耐心等待，不要按键盘，否则会退出。
